TenziesGame

React

GitHub Repo Live version

This is a recreation of the classic game Tenzies. This was the capstone project of the Scrimba Learn React Course. It was built fully using React.js with some CSS styling.

Props were passed into each Dice component to enable them to have the functionality of holding the dice at a specific value and storing that value. The event listener function holdDice was passed as a prop to avoid using Derived state and kept the state in the parent component.

UseState was another feature that was implemented in this project as a means of setting the dice, the rolls, and a boolean of whether the game was won or not.

UseEffect was also used as a way to keep two different pieces of state in sync when the game is won.

I implemented the extra feature where the number of rolls the user takes is tracked and displayed when the user wins the game.

BoxClickerGame

HTML, CSS, & JavaScript

GitHub Repo Live version

I was inspired to create this game from a game I played when I was younger. That game involved dragging a dot across the screen without touching any obstacles. I designed this game without obstacles and instead focused on the randomness of the objects appearing.

After pressing start, colorful boxes will randomly generate. Hovering over the boxes will increase the player’s score, which is saved to the user’s local storage to facilitate replay-ability.

I incorporated CSS custom properties to style the dark and light modes of the site.
